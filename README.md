# 🧠 AI-Powered Task Manager

A smart task management web app with authentication, real-time database, and AI-based suggestions.

## 🚀 Features
- Login/Signup (Firebase)
- Add & Delete Tasks
- Priority Management
- Real-time updates
- AI suggestions

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Firebase (Auth + Firestore)
